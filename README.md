# Прогноз опадів (Open-Meteo API)

Прототип сервісу для прогнозування погоди. Завантажує історичні метеодані, навчає три моделі машинного навчання, обирає найкращу та видає прогноз опадів на наступні 3 дні. Інтерфейс зроблено на Streamlit.

## Дані
Використовуються безкоштовні щоденні дані з Open-Meteo (історичні для навчання та прогнозні для результату).
Поля:
- precipitation_sum (сума опадів) - використовується лише для створення цільової змінної (якщо більше 0, то клас 1 - опади є, інакше 0 - опадів немає). Перед навчанням видаляється з ознак, щоб уникнути витоку даних.
- temperature_2m_max (максимальна температура)
- temperature_2m_min (мінімальна температура)
- windspeed_10m_max (швидкість вітру)

Після завантаження історичний датасет автоматично зберігається у файл weather_daily.csv.

## Моделі
Дані розбиваються на тренувальну та тестову вибірки (test_size=0.1). Усі ознаки проходять стандартизацію за допомогою StandardScaler.
Для класифікації навчаються три алгоритми:
1. Логістична регресія
2. Випадковий ліс (RandomForestClassifier)
3. Метод найближчих сусідів (KNeighborsClassifier)

Система порівнює їхню точність (Accuracy), обирає найкращу та зберігає її разом зі стандартизатором у файли .pkl за допомогою бібліотеки joblib.

## Структура проєкту
- app.py - скрипт застосунку Streamlit.
- weather_daily.csv - файл з історичними даними.
- best_model.pkl, scaler.pkl, features.pkl - збережені файли найкращої моделі та її налаштувань.
- requirements.txt - список бібліотек.
- README.md - опис проєкту.

## Запуск
1. Встановіть необхідні бібліотеки: pip install -r requirements.txt
2. Запустіть застосунок командою: streamlit run app.py
